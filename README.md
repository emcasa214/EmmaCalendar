## How To Setup
```
git clone https://github.com/emcasa214/EmmaCalendar.git
```
```
cd EmmaCalendar
```
```
source .env/bin/activate
```
```
python manage.py makemigrations
```
```
python manage.py migrate
```
```
python manage.py runserver
```
